# pyls - Python Implementation of Unix `ls` Command

## Overview

`pyls` is a Python-based implementation of the Unix `ls` command. It allows users to list directory contents with options for long format and file classification, similar to the Unix `ls` command.

## Features

- **Long Listing Format (`-l`)**: Displays detailed information about each file, including permissions, size, and modification time.
- **File Classification (`-F`)**: Classifies files with a symbol (e.g., `/` for directories, `*` for executables).
